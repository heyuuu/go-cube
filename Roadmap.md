# Roadmap

## 命令

### 快捷命令

- `clone`:  alias `project clone`
- `info`:   alias `project info`
- `search`: alias `project search`

### 全部命令

- [ ] `ws`: 工作区相关命令
  - [ ] `list`: 列出所有工作区
- [ ] `project` 项目相关命令
  - [ ] `search` 搜索项目列表
  - [ ] `info`   查看项目信息
  - [ ] `open`   打开项目
  - [ ] `clone`  克隆新项目
  - [ ] `temp`   管理临时项目
- [ ] `app` 应用管理
  - [ ] `list`   列出所有应用列表
  - [ ] `search` 搜索可用应用列表
- [ ] `repo` 源管理
  - [ ] `list` 源列表
